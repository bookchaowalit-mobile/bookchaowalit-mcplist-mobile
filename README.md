# Mcplist — Mobile

SwiftUI native iOS/macOS app for **Mcplist**.

Part of [Chaowalit Greepoke](https://bookchaowalit.com)'s 101 Portfolio Projects.

## Tech Stack

- **Framework:** SwiftUI (iOS 17+ / macOS 14+)
- **Language:** Swift 5.10
- **Architecture:** MVVM
- **Navigation:** NavigationStack + TabView

## Requirements

- Xcode 15+
- iOS 17+ / macOS 14+

## Getting Started

```bash
open Package.swift  # Opens in Xcode
# Or build from CLI:
swift build
```
